Vamos a jugar el juego de las escondidas (hide and seek): Se te ha dado un mapa de 10x10 celdas y en una de ellas hemos escondido el que será tu objetivo. Te puedes mover desde/hacia cualquier celda en el campo y en cada movimiento te informaremos si este te acerca o te aleja de tu objetivo, en comparación a tu ubicación anterior. Tu función deberá recopilar datos de pasos anteriores, como una lista de listas, donde los dos primeros elementos de cada lista indican tus coordenadas (fila y columna) y el tercer, si te encuentras más cerca o más lejos de tu objetivo (más frío o más caliente):

  • "más frío" es -1.
  • "más caliente" es 1.
  • "igual" es 0.

Para medir la distancia hasta tu objetivo, deberás usar la distancia euclídea. En cada paso debes devolver las coordenadas del siguiente. Si uno de tus pasos te coloca dentro de la celda que tiene objetivo ganas el juego. Deberás encontrar el objetivo usando un máximo de 12 pasos.

Datos de Entrada: Información sobre los pasos anteriores como una lista de listas. Cada lista contiene coordenadas e información de si te encuentras más cerca o más lejos de tu objetivo (caliente:warm, frío:cold, igual:same), con respecto al paso anterior.

Output: Las coordenadas del siguiente paso como una lista / tupla de dos enteros (int).

Ejemplo: 

checkio([[2, 2, 0]])  # [0, 2]
checkio([[2, 2, 0], [0, 2, -1]])  # [3, 2]
checkio([[2, 2, 0], [0, 2, -1], [3, 2, 1]])  # [4, 1]
checkio([[2, 2, 0], [0, 2, -1], [3, 2, 1], [4, 1, 0]])  # [3, 1]

¿Cómo se utiliza?: Usa este concepto para encontrar alguna cosa usando datos indirectos, como podría ser un sonido en un detector de metales o contador de radiación. Este es un problema de predicción que puede ser utilizado en machine learning.

Condiciones: |map| = 10x10
